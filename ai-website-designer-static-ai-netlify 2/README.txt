# AI Website Designer Static Share Version

This is a frontend-only static demo that can be uploaded to Netlify Drop.

## How to publish

1. Unzip this folder.
2. Go to Netlify Drop: https://app.netlify.com/drop
3. Drag the whole `ai-website-designer-static-share` folder into the drop area.
4. Netlify will give you a public URL like `your-site-name.netlify.app`.

## What works

- Website generator demo
- Specially Made demo
- Photo upload locally inside the browser
- 2D game generator demo
- Generated preview iframe
- Copy generated HTML

## What does not work in this static version

- Backend saving
- Real image storage
- Admin export
- Real payment
- Server-side AI calls

This is best for quickly sharing the concept with someone.

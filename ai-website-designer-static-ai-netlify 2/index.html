How to put this on a public domain

  1. Download and unzip this folder.
  2. Open Netlify Drop.
  3. Drag the whole folder into the drop area.
  4. Netlify will give you a public link like your-site.netlify.app.

This is the fastest way to share the demo. For the full backend version later, use Render or another Node host.
